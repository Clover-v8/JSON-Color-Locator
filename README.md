Universal JSON Color Locator – Pro
A lightweight web tool for extracting, visualizing, and batch-editing color values inside JSON files.
This tool recursively scans JSON objects and arrays to locate hex colors and transparent values, displays them in a visual grid, and allows selective or bulk replacement by exact JSON path. It includes undo support, live preview, and JSON export.
Features
Deep JSON traversal (objects & arrays)
Detects hex colors (#RGB, #RGBA, #RRGGBB, #RRGGBBAA) and transparent
Visual color grid with occurrence count
Path-level color replacement
Bulk select & batch editing
Undo history (Ctrl / Cmd + Z)
Dark / Light mode with persistence
Responsive UI (desktop & mobile)
Export updated JSON file
Use Cases
Theme and design token editing
UI color refactoring
JSON configuration cleanup
Large-scale color replacement without manual searching
Tech
Vanilla HTML, CSS, and JavaScript — no dependencies.
